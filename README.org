#+title: File-System Demo (PhantomOS)

A demo application showcasing file-system capabilities of WASM inside PhantomOS.

* Installation
** Pre-requisites
- wasi-sdk (*v22.0*)
- phantomuserland (with wasm-mirco-runtime submodule)
- make

  
** Building
Check the paths in the [[./Makefile][Makefile]] are correct, then run:

#+begin_src shell
  make
#+end_src

* Running
To run the application it should be launched from an already-running PhantomOS process. For example, you can launch it from _plib/sys/src/ru.dz/ru.dz.phantom.system.shell.ph_:

#+begin_src c
  // Add to imports
  import .ru.dz.demo.fs_demo;

  // Declare the class
  var fs_demo : .ru.dz.demo.fs_demo;

  // And launch!
  fs_demo = new .ru.dz.demo.fs_demo();
  fs_demo.run(console);
#+end_src

Then rebuild the PhantomOS classes:

#+begin_src shell
  make -C $PHANTOM_HOME/ classes-clean classes
#+end_src
